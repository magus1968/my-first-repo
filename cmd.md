# Подсказки по командной строке

#### Quit - клавиша позволяющая выйти из какого-то отображения
*PS: мне понадобилась после вывода большого количества log'a*
```sh
Q
```

#### Команда очистки терминала
```sh
clear
```

#### Показать текущую директорию - путь
```sh
pwd
```

#### Отобразить листинг текущей директории
```sh
ls
# Аналог [ls .]
# В отличии от [dir] вид папок будет визуально отличаться от вида файлов

ls ..
# Вывод листинга вышестоящей директории 
ls -l
# Вывод длинной (дополнительной) информации

dir
# Аналог [ls] - просто папки и файлы отображаются одинаково
```

#### Команда смены директории
```sh
cd folder_name
# В отличие от Windows в Git Bush обычный слэш /
# Используем TAB -- чтобы не вводить имя папки (файла) полностью

cd ..
# Быстрый переход на уровень вверх
cd ../..
# Переход на два каталога выше
cd
# Быстрый переход в домашнюю директорию
```

#### Псевдонимы для директорий
```sh
.   # текущая директория
..  # родительская директория
```

#### Создать папку с указанным именем
```sh
mkdir folder_name

mkdir folder_name/new_folder
# Создает вложенный каталог в [folder_name] находясь в текущем -- без необходимости перехода в [folder_name] 

mkdir folder_name && cd folder_name
# Две команды в одной строке: создание каталога и (&&) переход в него
```

#### Копирование (copy)
```sh
cp file_name.ext file_name_2.ext
# Создает копию файла в текущей директории

cp file_name.ext folder_name/
# Копирует файл во вложенную директорию

cp -r folder_name folder_name_2/
# Создает копию директории в текущем каталоге
```

#### Перемещение (move)
```sh
mv file_name.ext folder_name/
# Перемещает файл во вложенную директорию

mv folder_name folder_name_2/
# Перемещает в текущем каталоге: одну вложенную директорию в другую
```

#### Удаление файла (remove)
```sh
rm file_name

del file_name
# Аналог [rm] в Windows, у меня в Git Bush работает
```
 #### Удаление папки
 ```sh
rm -r folder_name
# Удаляет папку и всё её содержимое навсегда -- корзины нет

rm -rf .git
# Удаляет локальный репозиторий (на GitHub остаётся)
 ```

#### Вывести содержимое файла - *котануть* файл
```sh
cat file_name
# Читает файлы построчно
```

#### Подсчитать слова и строки в файле
```sh
wc file_name

wc -l file_name
# С ключом [-l] считает строки
```

#### Канал `|` *(вертикальная черта)* - передаёт вывод одной команды на вход другой
```sh
cat file_name | wc -l

# [wc] читает строки из вывода команды [cat]
```

#### Сортировка
```sh
sort

# По умолчанию интерпретирует то, что передано на вход, как массив строк и сортирует его в лексикографическом порядке
```

#### Удаление дубликатов
```sh
uniq

# ! Есть особенность: [uniq] удаляет дубликаты только из последовательностей одинаковых слов, идущих подряд -- поэтому файл вначале необходимо отсортировать
```

---
#### **Конвейер** - состыковка нескольких Unix-команд между собой через **канал**

```sh
 cat file_name.txt | sort | uniq | wc -l
 # Посчитали количество уникальных слов в файле с использованием 4-х команд
```
#### Команды для командной строки в Jupyter-ноутбуке
```sh
%pwd
%ls
%cd folder_name
%mkdir folder_name
%ls folder_name

# При работе в Jupyter-ноутбуке под Windows -- перед командой нужно прописать знак процента %. В macOS и Linux -- восклицательный знак !
```
